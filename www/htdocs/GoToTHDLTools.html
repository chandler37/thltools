THDL Tools at SourceForge

Welcome to the home of software tools for the Tibetan and Himalayan Digital Library (THDL).  Here you can find Jskad, QuillDriver, Savant, the Tibetan Machine Web fonts, and other useful pieces of software.

More information about these tools can be found at the THDL's Tools site.   Please refer to that site first, and come back here only if you don't find what you're looking for.

The only real reason to visit these web pages, rather than the THDL's much more useful pages, is to download old versions or alpha versions of the software (note that the current releases are available for download from the main THDL Tools site), to submit bug reports or feature requests, to help contribute to the project, etc. If you're trying to do any of these things, we hope you'll understand why we haven't spent any time making this site very good-looking or easy to use.  Instead, you'll have to see our developers' site or find your way through the maze of options available at the THDLTools project page.   If the above description doesn't match you, and you feel that you've had to wade through the impenetrable project page for something that the main THDL Tools site should have provided, please let us know of your experience so that we can make the THDL tools more accessible.

Once again, your two best bets for information related to the THDL software tools are:

  • the main THDL Tools site (a good-looking, easy-to-use site)
  • the THDLTools project page at SourceForge (a very technical, hard-to-use site)
